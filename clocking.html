Employee Clocking

Time management logic aligned with the functional rules: workday starts at 08:00, regular hours end at 17:00, first-task delay becomes waste time, idle gaps become waste time, and work after 17:00 becomes overtime.
Active Rules
  • First task delay from 08:00 is auto-recorded as waste time.
  • Work Type, Machine, Order Number, and Task Description are mandatory for productive work.
  • After ending a task, if no new task starts, a waste session can be opened immediately.
  • Overtime is counted only after 17:00 and is stored separately.
Use Waste Time for waiting, break, idle, cleaning, rework, etc.
